# Tic Tac Toe Pro

An interactive Tic Tac Toe Web Application built using HTML, CSS, and JavaScript.

## Features

* Player vs Player Mode
* Player vs Computer Mode
* Custom Player Names
* Real-Time Scoreboard
* Draw Counter
* Games Played Tracker
* Match History
* Winner Popup
* Confetti Celebration
* Dark/Light Mode Toggle
* Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Learning Outcomes

This project helped strengthen my understanding of:

* DOM Manipulation
* Event Handling
* Game Logic
* Responsive Web Design
* Front-End Development
